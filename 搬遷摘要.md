## 本機步驟

```powershell
# 1. 裝 Hermes
irm https://hermes-agent.nousresearch.com/install.ps1 | iex

# 2. 匯入狀態（skills、cron、memory、session 全包）
hermes profile import .\hermes-profile-default.tar.gz

# 3. 設 API key
echo DEEPSEEK_API_KEY=*** >> $env:USERPROFILE\.hermes\.env

# 4. 拉專案
git clone <repo-url> ~/hermes-agent

# 5. 確認
hermes doctor
hermes config set model.provider deepseek
hermes config set model.model deepseek-v4-pro
```
